# Siddhesh Gadade — Premium Developer Portfolio

Welcome to my personal developer portfolio! This project is a modern, highly responsive, and beautifully animated web application built to showcase my backend expertise, UI/UX sensibilities, and selected projects.

![Portfolio Preview](./public/preview.png)

## Tech Stack
- **Framework**: [React 18](https://react.dev/) inside [Vite 6](https://vite.dev/)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Tokyo Night Custom Theme)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Features
- ✨ **Premium Design Aesthetic**: Strict adherence to the exquisite "Tokyo Night" theme (Glass-morphism, dark gradients, neon accents).
- 🚀 **Buttery Smooth Transitions**: AnimatePresence handles page mounting and unmounting for native-app like routing transitions.
- 📱 **Fully Responsive**: Flawless execution across mobile, tablet, and desktop viewports.
- 🎨 **Dynamic Atmosphere**: Subtly glowing, randomly animated background orbs render continuously behind the UI.
- ⚡ **Blazing Fast**: Vite powers instantaneous HMR during dev and heavily optimized assets during production builds.

## Getting Started

To run this project locally, simply clone the repository and run:

```bash
# Install dependencies
npm install

# Start the Vite development server
npm run dev
```

Visit `http://localhost:5173` to see the app running live.

## Deployment Status
Optimized and ready to be deployed to **Vercel** with zero-configuration required. All client-side routing is inherently handled by their Edge network routing.
